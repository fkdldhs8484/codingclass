1부터 100까지 출력하기

1.1 while문을 이용해서 1부터 100까지 출력하시요!

{
    let num = 1;
    while(num <= 100){
        document.write(num);
        num++;
    }
}

{
    for(let i=1; i<=100; i++){
        document.write(i)
    }
